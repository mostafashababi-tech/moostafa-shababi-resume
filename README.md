# فروشگاه لوازم یدکی خودرو

یک فروشگاه آنلاین کامل برای فروش لوازم یدکی خودروها با امکانات زیر:

## ویژگی‌ها

- نمایش محصولات با امکان فیلتر بر اساس دسته‌بندی و برند
- جستجوی پیشرفته محصولات
- صفحه جزئیات محصول با اطلاعات کامل
- سیستم سبد خرید کامل
- احراز هویت کاربران (ثبت‌نام و ورود)
- پنل مدیریت برای افزودن و ویرایش محصولات
- طراحی ریسپانسیو و زیبا
- پشتیبانی از زبان فارسی (RTL)

## تکنولوژی‌های استفاده شده

### فرانت‌اند
- React 18
- Vite
- React Router
- TailwindCSS
- فونت Vazirmatn

### بک‌اند و دیتابیس
- Supabase (PostgreSQL)
- Supabase Auth برای احراز هویت
- Row Level Security (RLS) برای امنیت

## نصب و راه‌اندازی

### 1. نصب وابستگی‌ها
```bash
npm install
```

### 2. تنظیم متغیرهای محیطی
فایل `.env` را در ریشه پروژه ایجاد کنید و متغیرهای زیر را تنظیم کنید:

```env
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 3. راه‌اندازی دیتابیس
دیتابیس Supabase به صورت خودکار از طریق migration ایجاد شده است و شامل موارد زیر است:
- جداول: brands, car_models, categories, products, profiles, cart_items, orders, order_items
- سیاست‌های RLS برای امنیت
- داده‌های نمونه (8 برند، 8 دسته‌بندی، 10 محصول)

### 4. ایجاد کاربر ادمین
برای دسترسی به پنل مدیریت:

1. در سایت ثبت‌نام کنید
2. در Supabase Dashboard بروید به جدول `profiles`
3. فیلد `is_admin` کاربر خود را به `true` تغییر دهید

### 5. اجرای پروژه

```bash
npm run dev
```

### 6. بیلد برای Production

```bash
npm run build
```

## امکانات پنل مدیریت

پنل مدیریت شامل سه بخش اصلی است:

1. مدیریت محصولات - افزودن، ویرایش و حذف محصولات
2. مدیریت دسته‌بندی‌ها - افزودن و ویرایش دسته‌بندی‌ها
3. مدیریت برندها - افزودن و ویرایش برندهای خودرو

## امنیت

- تمام جداول دارای Row Level Security (RLS) هستند
- فقط کاربران احراز هویت شده می‌توانند به سبد خرید دسترسی داشته باشند
- فقط ادمین‌ها می‌توانند محصولات را مدیریت کنند
- تمام ورودی‌های کاربر اعتبارسنجی می‌شوند
